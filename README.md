# NLA1 – Numerical Linear Algebra Project

## Description

This repository contains the first semester project for the Numerical Linear Algebra (NLA1) course at VSB – Technical University of Ostrava.

The project focuses on the implementation and analysis of selected numerical linear algebra algorithms. Both theoretical derivations and computational verification were performed using Python and Jupyter Notebook.

## Project Tasks

### Task 1 – Complex Matrix Multiplication

Implementation of an efficient algorithm for multiplying complex matrices using only three matrix multiplications instead of the standard four.

Topics covered:

* Complex matrix arithmetic
* Computational optimization
* Matrix algebra

### Task 2 – Householder Transformation

Construction of a Householder matrix that transforms one vector into another.

Topics covered:

* Orthogonal transformations
* Householder reflections
* Matrix factorization concepts

### Task 3 – Conjugate Gradient Method

Manual implementation and analysis of the Conjugate Gradient (CG) iterative method for solving a system of linear equations.

Topics covered:

* Symmetric positive definite matrices
* Iterative numerical methods
* Convergence analysis
* Error estimation
